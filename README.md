# ML Algorithms

This repository contains a collection of classic machine learning algorithms implemented in Python using popular libraries such as `numpy`, `pandas`, and `scikit-learn`.

## Algorithms Included

- Linear Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes
  - Bernoulli Naive Bayes
  - Gaussian Naive Bayes
  - Multinomial Naive Bayes
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)

## Dataset Folder

The `data/` directory includes sample datasets for experimentation and model training, such as:

- `iris.csv`
- `titanic.csv`
- `housing.csv`
- `WineQT.csv`
- `Breast_cancer_wisconsin.csv`
- `ionosphere/`
- `mushroom/`
- `sms+spam+collection/`

## Run an Example

You can run any script directly from the project root:

```bash
python Linear_Regression.py
python Logistic_Regression.py
python KNN.py
```

## Requirements

Install the required packages using:

```bash
pip install numpy pandas scikit-learn matplotlib seaborn
```

## Notes

Each script is a standalone example and can be used for learning, experimentation, and model comparison.
